# WPF 원그리기

삼각함수를 사용해 x, y 좌표를 구한 후 원을 그리는 방법을 알아봅시다.

<br>

## 삼각함수 공식
<img src="https://user-images.githubusercontent.com/68521148/145054110-4aaf0d61-41d9-4711-ae24-b535efa27ebb.png" width="400" height="400"></img>

**@ = 각도, r = 반지름, a = 높이, b = 밑변**

```
Sin@: r / a => a(y좌표): Sin@ * r
Cos@: r / b => b(x좌표): Con@ * r
```
