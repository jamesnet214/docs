## SPA 란?
`Single Page Application`의 약자로 페이지가 1개인 어플레케이션이란 뜻입니다.    
웹 실행에 필요한 모든 정적 리소스를 최초에 한번만 다운로드 후 새 페이지(html)를 불러오지 않고 현재 페이지를 동적으로 다시 작성합니다.

## SPA가 생겨난 배경
기존 웹사이트(Multiple Page Application)방식은 지금보다 전달되는 페이지(html)의 용량이 작았다. 그래서 새로운 페이지를 서버에서 전송(Server Side Rendering) 해 주었는데 점차 웹사이트가 고도화되며 페이지의 용량이 커져가 매번 새로운 페이지를 전달하는 것이 버거워지게 되어 SPA가 등장하게 되었다
